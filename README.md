# Web-Based Sudoku Game for Seniors

A simple, accessible Sudoku game designed specifically for senior users, with Japanese as the default language.

## Features

- Large, high-contrast numbers for easy visibility
- Simple click-based interface
- Annotation mode for notes and solving aids
- Japanese/English language support
- Responsive design for tablets and desktop
- No timers or stress-inducing elements

## How to Use

1. Open `index.html` in a modern web browser
2. Select difficulty level and click "新しいゲーム" (New Game)
3. Click cells to select them, then click numbers to fill
4. Use "メモモード" for annotations and "消去" to erase
5. Complete the puzzle to see the congratulatory message

## Browser Requirements

- Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+
- Works on desktop and tablet devices
- No installation or internet connection required

## Development

This is a single-file web application using vanilla HTML, CSS, and JavaScript with no external dependencies.
