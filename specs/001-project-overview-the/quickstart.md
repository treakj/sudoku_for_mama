# Quickstart Guide

## Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required

## Running the Application
1. Open `index.html` in your web browser
2. The game loads with Japanese as default language
3. Select difficulty and start playing

## Basic Usage
1. Click a cell to select it (highlighted with blue border)
2. Click a number on the pad to place it
3. Use "メモモード" for annotations
4. Use "消去" to clear entries
5. Switch language with the dropdown if needed

## Testing Scenarios
- **New Game**: Click "新しいゲーム", select difficulty
- **Annotation**: Toggle mode, click numbers for notes
- **Highlighting**: Select a number or cell to see highlights
- **Win Condition**: Complete the puzzle to see congratulatory message

## Browser Compatibility
- Desktop: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- Tablet: iPad Safari, Android Chrome
